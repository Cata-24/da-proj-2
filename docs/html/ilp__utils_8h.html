Detailed Description

Defines the ILP class that manages the Integer Linear Programming approach to solve the knapsack problem, including input preparation, running the solver, and output parsing.

This class handles:

  • Writing input data to a file for the ILP solver.
  • Executing an external Python ILP solver script.
  • Reading and parsing the solver output back into the program.
