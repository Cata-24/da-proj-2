This file declares two main structs\+: {\ttfamily \doxylink{structinput__data}{input\+\_\+data}} and {\ttfamily \doxylink{structoutput__data}{output\+\_\+data}}.

\item {\ttfamily \doxylink{structoutput__data}{output\+\_\+data}} stores the result of the algorithm.
