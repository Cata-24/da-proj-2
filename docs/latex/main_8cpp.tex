\doxysubsection{Detailed Description}
Main entry point for the Delivery Truck \doxylink{classPallet}{Pallet} Packing Optimization application. 

This program provides a command-\/line interface that using various algorithmic approaches solves the pallet packing problem. It repeatedly gathers user input, runs the selected algorithm on the input data, measures execution time, and displays the results.

The main loop allows continuous interaction until the program is terminated manually.
