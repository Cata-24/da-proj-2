\doxysubsection{Detailed Description}
Provides utility functions for loading and parsing input data from CSV files. 

This file includes functions to load pallet\textquotesingle{}s id, weight and profit and truck\textquotesingle{}s capacity from CSV datasets. It also provides the {\ttfamily startup} function to parse the {\ttfamily \doxylink{structinput__data}{input\+\_\+data}} structure used in knapsack problem solvers.
