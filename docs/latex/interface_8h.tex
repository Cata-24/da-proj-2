\doxysubsection{Detailed Description}
Defines the \doxylink{classInterface}{Interface} class for terminal-\/based user interaction. 

This class handles user input needed to select datasets and algorithms, and displays the results after the algorithm execution. It contains all the functionality related to input/output operations in the console for the Delivery Truck \doxylink{classPallet}{Pallet} Packing Optimization problem.
